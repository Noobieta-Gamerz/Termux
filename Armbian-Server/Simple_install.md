##### Running Armbian Server
```
cd /sdcard/Download
mv armbian-server-termux.tar.xz $HOME
mkdir armbian-server
mv armbian-server-termux.tar.xz armbian-server
cd armbian-server
tar -xvf armbian-server-termux.tar.xz
chmod +x start-armbian.sh
./start-armbian.sh
or
bash start-armbian.sh
```
